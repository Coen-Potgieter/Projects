- This will tackle the MNIST Data set
- This means we will have a 28x28 pixel image

Specifications:
- Layer1
    - Convolution
        - Filters: 2
        - Filter size: 5 (radius=2)
        - Activation Function: ReLU
        - This will output 2, 24x24 images

    - Pooling
        - Type: Max Pooling
        - Pool Size: 2x2
        - Stride: 2
        - This will produce 2, 12x12 images 

- Layer2
    - Convolution
        - Filters: 4
        - Filter Size: 3x3x2
        - Activation funciton: sigmoid
    
    - Pooling 
        - Type: Max Pooling
        - Size: 2x2
        - Stride: 2
